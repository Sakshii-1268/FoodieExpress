# FoodieExpress
Food Ordering Web App 🍔🍕
A full-stack web application built with the MERN stack (MongoDB, Express, React, Node.js) and TypeScript. This app allows users to browse a menu, place food orders, and manage their account.

Features
✅ User authentication (sign up, login)
✅ Browse and search for food items
✅ Add items to the cart and place orders
✅ Admin interface to manage food items
✅ Order tracking and history

Tech Stack
Frontend: React, TypeScript, Axios
Backend: Node.js, Express, TypeScript
Database: MongoDB
Styling: CSS, Bootstrap
